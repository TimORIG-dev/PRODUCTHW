# Угадай число (консольная игра)

Простая консольная игра на Python без внешних зависимостей.

## Правила

Программа загадывает случайное целое число от 1 до 100. Вы вводите варианты, а игра подсказывает:

- `Больше` — если загаданное число больше вашего
- `Меньше` — если загаданное число меньше вашего

После угадывания игра показывает количество попыток и предлагает сыграть заново.

## Требования

- Python 3.8+ (подойдёт и новее)

## Запуск

Из корня проекта:

```bash
python game.py
```

На некоторых системах может понадобиться:

```bash
python3 game.py
```

## Пример сессии

```text
Я загадал число от 1 до 100. Попробуйте угадать!
Ваш вариант: 50
Больше
Ваш вариант: 75
Меньше
Ваш вариант: 63
Угадали! Количество попыток: 3
Сыграть ещё раз? (y/n): n
Спасибо за игру!
```
